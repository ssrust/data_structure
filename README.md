# data_structure
## Branch 사용 규칙
- 새 자료구조 구현
  - 이름: dev_${자료구조명}
  - Branch from: main
  - 완료 이후: Latest main commit에 merge
- Issue 수정
  - 이름: issue_${Issue No.}
  - Branch from: Issue 발생한 브랜치
  - 완료 이후: 해당 issue가 유효한 last commit에 merge
